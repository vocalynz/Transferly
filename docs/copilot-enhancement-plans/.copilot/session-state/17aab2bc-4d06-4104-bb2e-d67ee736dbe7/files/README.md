## 🎨 Design System Overview

### Color Palette
```
Primary:    #f8812d (brand orange)
Success:    #22c55e (emerald)
Warning:    #f59e0b (amber)
Error:      #ef4444 (red)
Info:       #0ea5e9 (sky blue)
Neutral:    #64748b (slate-500)
```

### Typography
```
Display:    Poppins (bold, modern)
Body:       Inter (clean, readable)
Mono:       Fira Code (technical)
```

### Spacing Scale
```
xs: 4px    sm: 8px    md: 12px   lg: 16px
xl: 20px   2xl: 24px  3xl: 28px  4xl: 32px
```

### Shadow Hierarchy
```
xs-glass:  Subtle (cards, inputs)
sm-glass:  Light (hovered states)
md-glass:  Medium (elevated content)
lg-glass:  Strong (modals)
xl-glass:  Premium (key UI)
```
